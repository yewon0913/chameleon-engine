# 카멜레온 엔진 (Chameleon Engine)

## 프로젝트 개요
소상공인 마케팅/콘텐츠/CRM 통합 도구. 카멜레온 브랜드.
원스톱 자금 AI(onestop-engine)와 같은 Supabase DB 공유.

- GitHub: https://github.com/yewon0913/chameleon-engine
- 원스톱 엔진: https://github.com/yewon0913/onestop-engine-

## 기술 스택
- Next.js 15 + TypeScript + Tailwind CSS
- tRPC
- Drizzle ORM + Supabase PostgreSQL (Seoul region)
- Claude API (Anthropic)
- Vercel 배포

## 디자인 시스템
- 메인 액센트: #FF6B35 (오렌지)
- 배경: 다크 모드 (#0a0a0a ~ #111827)
- 모바일 퍼스트 반응형

## 환경변수
- ANTHROPIC_API_KEY, DATABASE_URL
