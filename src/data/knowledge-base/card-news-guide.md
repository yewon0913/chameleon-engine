# 카드뉴스 제작 완전 가이드

## 3단계 제작법

### Step 1: 스크립트 작성
- 주제 선정 → 타겟 확인 → 장수 결정 (5~10장)
- 1장: 후킹 표지 (질문/숫자/충격)
- 2~N-1장: 핵심 내용 (1장 1메시지)
- 마지막장: CTA (팔로우/저장/링크)

### Step 2: 디자인 분석
- 경쟁 계정 상위 게시물 분석
- 컬러/폰트/레이아웃 벤치마킹
- 브랜드 톤 일관성 확인

### Step 3: 피그마/캔바 제작
- 1080×1080px (정사각형) 또는 1080×1350px (4:5)
- 템플릿 기반 반복 제작
- 텍스트→이미지→정렬→검수

## 4대 원칙

### 1. 후킹 표지
- 숫자: "99%가 모르는", "5가지 방법"
- 질문: "왜 안 될까요?"
- 충격: "이거 실화?"
- 폰트 크기: 제목 40pt+ (모바일 가독성)

### 2. Hook → 핵심 → CTA
- 1장: Hook (멈추게 하기)
- 2~N장: 핵심 정보 (가치 전달)
- 마지막: CTA (행동 유도)

### 3. 1장 1메시지
- 한 장에 메시지 1개만
- 텍스트 과다 금지
- 리스트형: 1~2문장
- 서술형: 3~4문장 최대

### 4. 디자인 규칙
- 컬러 3색 이내 (메인+서브+포인트)
- 여백 충분히 (답답함 금지)
- 정렬 일관성 (좌정렬 또는 중앙정렬 통일)

## 레이아웃 패턴 7종

| # | 패턴 | 구성 | 적합한 콘텐츠 |
|---|------|------|-------------|
| 1 | 텍스트 중심 | 배경색+큰 텍스트 | 명언/팁/후킹 |
| 2 | 이미지+텍스트 | 상단 이미지+하단 설명 | 제품/음식 |
| 3 | 아이콘+리스트 | 아이콘 3~4개+설명 | 비교/장점 |
| 4 | 숫자 강조 | 큰 숫자+부가 설명 | 통계/실적 |
| 5 | Before/After | 좌우 또는 상하 분할 | 변화/효과 |
| 6 | 타임라인 | 세로 흐름 | 절차/과정 |
| 7 | 인용/후기 | 따옴표+이름 | 사회적 증거 |

## 스크립트 작성 규칙

### 규칙 1: 구어체 사용
- "~입니다" → "~거든요", "~잖아요"
- 대화하듯 자연스럽게

### 규칙 2: 숫자 먼저
- "매출이 올랐습니다" → "매출 300% 올랐습니다"
- 구체적 숫자가 신뢰도 상승

### 규칙 3: 행동 유도 명확
- "자세한 내용은..." → "지금 프로필 링크 클릭!"
- CTA는 직접적으로

## 디자인 상세

### 폰트
- **제목**: Pretendard Bold (40~60pt)
- **본문**: Noto Sans KR Regular (20~28pt)
- **강조**: Pretendard ExtraBold + 포인트 컬러
- 폰트 2종 초과 금지

### 여백 비율
- 상단: 15% (로고/카테고리)
- 중앙: 70% (핵심 콘텐츠)
- 하단: 15% (CTA/페이지번호)

### 텍스트 밀도
- 리스트형: 항목당 1~2문장 (20자 이내)
- 서술형: 3~4문장 최대 (80자 이내)
- 빈 공간 > 텍스트 (답답함 방지)

### 컬러
- 메인: 브랜드 컬러 1개
- 서브: 메인의 밝은/어두운 톤
- 포인트: 대비색 (CTA 버튼용)
- 배경: 화이트 또는 연한 파스텔

### 가독성 체크
- 모바일 피드에서 축소 시 텍스트 읽히는지
- 배경과 텍스트 대비 4.5:1 이상
- 이모지는 포인트로만 (남용 금지)
