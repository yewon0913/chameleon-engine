## 채널 1: 정책자금연구소 바름

| 항목 | 내용 |
|------|------|
| 추정 구독자 | 1만~3만 (정책자금 전문 니치) |
| 주력 콘텐츠 | 소상공인/중소기업 정책자금 심사 꿀팁, 사업계획서 작성법, 대출 승인 사례 |
| 수익 구조 | **컨설팅 수임료** (메인) + 유튜브 광고 + 교육 프로그램 |
| 웹사이트 | lab-bareum.com |
| 벤치마킹 | 유튜브를 리드 생성 채널로 활용 → 컨설팅 전환. 전문성으로 신뢰 구축 |

## 채널 2: 일퍼센트 경영지원연구소 (1percent)

| 항목 | 내용 |
|------|------|
| 추정 구독자 | 1만~5만 |
| 주력 콘텐츠 | 정책자금 승인 사례, 신용등급별 전략, 업종별 맞춤 가이드, 부결 후 재도전 전략 |
| 수익 구조 | **컨설팅** (계약 5천건+, 누적 승인 500억+) + 유튜브 광고 |
| 웹사이트 | 1-percent.co.kr |
| 벤치마킹 | "승인율 99.7%" 같은 실적 기반 마케팅. 구체적 숫자로 신뢰 구축 |

## 채널 3: 사업하는형

| 항목 | 내용 |
|------|------|
| 추정 구독자 | 5만~10만+ |
| 주력 콘텐츠 | 소상공인 창업/경영 전반, 정책자금 활용법, 세무/노무 실무 팁 |
| 수익 구조 | 유튜브 광고 + 강의/세미나 + 협찬 + 멘토링 |
| 벤치마킹 | 넓은 타겟(사업 전반) → 높은 구독자 확보 → 정책자금 콘텐츠로 전문성도 커버 |

## 채널 4: 돈이되는세금 (세무 + 정책자금 융합)

| 항목 | 내용 |
|------|------|
| 추정 구독자 | 3만~10만 |
| 주력 콘텐츠 | 세금 절세 전략, 정책자금과 세무의 연결고리, 소상공인 절세+자금 동시 해결 |
| 수익 구조 | 세무사 사무소 연계 + 유튜브 광고 + 강의 |
| 벤치마킹 | 세금이라는 대중적 주제 + 정책자금 = 시너지. 세무사 본업과 유튜브 연결 |

## 채널 5: 소진공/중기부 공식 채널 + 정부 정책 해설 채널

| 항목 | 내용 |
|------|------|
| 추정 구독자 | 1만~5만 (공식 채널 기준) |
| 주력 콘텐츠 | 정책자금 공고 해설, 신청방법 안내, 성공사례, 현장 간담회 |
| 수익 구조 | 정부 홍보 목적 (비수익) |
| 벤치마킹 | 공식 정보 신뢰성. 이 콘텐츠를 쉽게 풀어주는 2차 해석 채널이 기회 |

## 정책자금 유튜브 시장 종합 분석

### 시장 특성
- **니치하지만 고객 가치가 높음:** 대출 1건당 수천만~수억 → 컨설팅 수임료 높음
- **신뢰가 핵심:** "돈" 관련이라 신뢰 구축이 무엇보다 중요
- **실적 마케팅이 효과적:** "승인 500억", "승인율 99.7%" 같은 숫자
- **유튜브 → 컨설팅 전환이 주요 수익원:** 광고 수익은 부차적

### 우리가 벤치마킹할 포인트
1. **AI 도구 + 콘텐츠 결합:** 슈가AI처럼 도구를 제공하면서 콘텐츠로 유입
2. **실제 사례 기반 콘텐츠:** "이 음식점이 이렇게 해서 승인받았다" 스토리텔링
3. **구체적 실적 숫자로 마케팅:** 승인 건수, 승인율, 누적 금액 공개
4. **세분화된 키워드 전략:** "카드론 3건 음식점 소진공" 같은 롱테일 키워드
5. **교육 → 도구 → 컨설팅 퍼널:** 무료 교육으로 유입 → AI도구로 체험 → 유료 컨설팅
