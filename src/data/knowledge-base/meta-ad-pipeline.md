# 메타광고 리드 수집 파이프라인 (안대장 자료 기반)

## 전체 흐름
```
메타 비즈니스 계정 → 광고 설정 → DB 자동화(Zapier) → CRM 연결 → 퍼널 자동화
```

## 1단계: 메타 비즈니스 계정 생성
- URL: https://business.facebook.com/
- Facebook 로그인 → 계정 만들기 → 비즈니스 이름 + 이메일 설정
- 이메일로 발송된 액세스 링크 확인 필수!
- 광고자산 생성: 비즈니스 설정 → 광고 계정 → 생성

## 2단계: 메타 광고 설정 (잠재고객 DB 수집용)

### 캠페인
- 목적: 잠재고객 (리드 수집)
- 수동 잠재고객 캠페인 선택

### 광고 세트
- 전환 위치: 인스턴트 양식 (핵심!)
- 일일 예산: ₩20,000부터 시작 권장
- 타겟 위치: 대한민국 / 지역별 설정 가능
- 최소 연령: 18세 이상 (고객 연령 고려)
- 노출 위치: 수동 설정 → Facebook + Instagram 체크

### 광고
- 미디어: 이미지/영상 업로드
- 문구: 팩트 + 범위 + 대상 + 다음 행동
- CTA: "지금 신청하기"
- 인스턴트 양식 만들기:
  - 양식 이름 설정
  - "더 많은 제출 수" 선택
  - 언어: 한국어
  - 공유: 공개
  - DB 유형 선택 (이름/전화번호/이메일 등)
  - 개인정보처리방침 링크 필수!
  - 양식 종료 시 "비즈니스에 전화 걸기" 설정

### 전환율 높은 광고 문구 패턴
- 리드수집형: "○○사업자 대상, 1분 만에 가능 여부 확인"
- 신뢰형: "누적 상담 N건 경험, 조건에 맞는 방향 안내"
- 전환형: "가능성을 높일 수 있는 시나리오부터 검토"
- 부담완화형: "복잡한 서류 전에, 가능한지부터 가볍게 체크"
- ⚠️ 금지: 1위/최고/무조건/보장 같은 단정형

## 3단계: DB 자동화 (Zapier + 솔라피)

### 사전 준비
1. Zapier 회원가입 + 유료 플랜 결제
2. 카카오톡 비즈니스 채널 개설
3. 솔라피 가입 + 카카오톡 채널 연동
4. 솔라피에 카카오톡 알림톡 템플릿 등록 + 검수

### Zapier 설정
- Create Zap
- 트리거: Facebook Lead Ads → New Lead
  - 페이스북 계정 연결 (권한 모두 동의)
  - 페이지 연결
  - Form: Default (Any Form)
  - 테스트 트리거 실행
- 액션 1: Google Sheets → Create Spreadsheet Row
  - DB 자동 적재
- 액션 2: 솔라피 → 카카오톡 알림톡 자동 발송
  - 신규 리드에게 자동 안내 메시지

### 자동화 흐름
```
고객이 메타 광고 양식 제출
→ Zapier가 감지 (트리거)
→ 구글시트에 DB 자동 적재 (액션1)
→ 솔라피로 카카오톡 알림톡 발송 (액션2)
→ CRM에 자동 등록
```

## 4단계: 잠재고객 DB 확인
- 광고관리자 → 결과 → Facebook 내 잠재고객 클릭
- 기간 설정 → 다운로드 (엑셀)
- 엑셀에서 이름/전화번호/이메일 등 DB 확인

## 카멜레온 연결 포인트

### intake (고객 유입)
- 메타광고 → 인스턴트 양식 → Zapier → CRM 자동 등록
- intake 폼 URL을 메타광고 CTA에 연결 가능

### funnel (퍼널 자동화)
- Day 1: 감사 메시지 (카카오톡 알림톡)
- Day 3: 무료 샘플/진단 제안
- Day 7: 관련 정보/사례 공유
- Day 14: 본격 제안/견적
- Day 30: 후기 요청

### CRM
- 메타광고 리드 → 자동으로 "문의" 상태로 등록
- 상담중 → 견적발송 → 계약완료 파이프라인

### autopilot
- 신규 리드 알림: "오늘 메타광고에서 N명 유입"
- 미응답 리드 알림: "3일째 미응답 리드 N명"
- 전환율 리포트: "이번 주 전환율 N%"

## 비용 구조
- 메타광고: 일일 2~5만원 (월 60~150만원)
- Zapier: 월 $49 (Professional 플랜)
- 솔라피: 건당 알림톡 비용 (건당 ~8원)
- 총 월 예상: 80~200만원
