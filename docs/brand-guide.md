# 카멜레온 브랜드 가이드

## 브랜드 보이스
- 톤: 트렌디하고 실용적
- 키워드: 자동화, AI, 수익화, 성장
- 금지: 기술 용어 남발, 비현실적 약속

## 색상
- 메인: 블랙 #0A0A0A + 골드 #D4AF37
- 그라디언트: 보라→에메랄드→골드
- 배경: #0A0A0A (다크 럭셔리)
- 텍스트: #F5F5F5 (밝은 회색)

## 고객 페르소나
- 이름: 박대표 (30~40대 1인 사업자)
- 페인포인트: "마케팅 할 시간이 없어"
- 니즈: "AI가 알아서 콘텐츠 만들어줘"
- 기대: "올리면 바로 반응 오는 콘텐츠"

## 폰트
- 제목: Pretendard Bold
- 본문: Pretendard Regular
- 코드: JetBrains Mono
